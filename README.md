# 🤖 Chatbot using Transformers

## 📌 Overview
This project is a conversational AI chatbot built using Hugging Face Transformers (DialoGPT).  
It interacts with users in natural language and generates context-aware, human-like responses.

## 🚀 Features
- Uses pre-trained transformer model (DialoGPT)
- Interactive console-based chatbot
- Maintains conversation context
- Dynamic response generation (no hardcoding)
- Exit using `exit` or `quit`

## 🛠️ Tech Stack
- Python
- Hugging Face Transformers
- PyTorch

## ▶️ How to Run
pip install transformers torch  
jupyter notebook chatbot_transformers.ipynb
